VITacademics Backend

VITacademics is an app for managing academics in the Vellore Institute of Technology, Vellore as well as Chennai campus.

This is our backend, and it runs here on Google App engine, acting as a custom built API for our client applications on Android, iOS, Windows Phone and even a Webapp.

Our backend application extracts data from the official university website.

Also, having one backend for all our client platforms helps us do the data extraction in a single language (Python, in this case) instead of writing the same code in multiple languages (Objective-C, Java, C#, Javascript) in each of the platforms.

API reference:

  • /captcha/(regno): Returns the bitmap of the captcha image
  • /captchasub/(regno)/(dateOfBirth)/(captcha): Submits and logs into academics with the given credentials
  • /attj/(regno)/(dateOfBirth): Returns the Attendance in JSON format
  • /marks/(regno)/(dateOfBirth): Returns the Marks in JSON format

The Future

This service has been designed such that a host of other services can be built on top of it. Some of the ideas we are exploring are:

  • Allow the college/faculty to send notices to the student's mobile device directly. Either via SMS or an App Notification
  • We look to build an event platform on top of this, allowing for a centralised portal for all the events that take place around college. Registration and immediate feedback from a cell phone would be very handy.
  • A service to help find other students who are travelling so as to allow sharing of cabs/autos to save money.
  • Most importantly, we could potentially provide a public API to allow other student developers to leverage our platform and make their own services on top of it. The possibilities are limitless.
